Scripts

lcc_mkdir

I enjoy collecting scans of old manuscripts, books and literary works. After some time I found that my collection was becoming increasingly unsorted. In order to combat this, I resolved myself to using the Library of Congress Classification system - a system similar to the Dewey Decimal System employed by the U.S. Library of Congress. This script pulls down the latest version of LCC categories/subcategories and creates a directory hierachy to aid in sorting. It is written in bash.

gogen

This simple command-line utility is meant to be used as a quick interface to the site (put in the Kanji), a Japanese etymological dictionary. When learning a new language it is often useful to know the origin of words, as it can aid memorization. This project can be found on my GitHub here.

Project Euler Framework
